# Business Performance Dashboard (Tableau)

## Objective
Create an interactive dashboard to analyze sales performance, customer behavior, and business trends.

## Tools Used
- Tableau
- Data visualization techniques

## Dataset
The dataset contains sales transactions including:
- customers
- products
- categories
- regions
- dates
- revenue

## Dashboard Features
- Total sales KPI
- Sales by customer
- Sales by category
- Sales by region
- Sales over time
- Interactive filters by region and category

## Key Insights
- Revenue is highly concentrated in a few customers, especially Ana.
- Electronics is the dominant category.
- East is the strongest region.
- Sales peaks are driven by high-value transactions.

## Business Value
The dashboard allows users to explore sales performance interactively and supports business decision-making by highlighting key performance drivers.
